평균이 틀렸을 때: 집단 분석에서 개인차 분석으로의 전환

2025 | 서울대학교 발달인지신경과학 연구실

배경

사람이 맥락이 바뀌는 시점을 인식할 때 동공이 확장된다는 선행연구를 바탕으로, 같은 맥락으로의 전환과 다른 맥락으로의 전환 간 동공 확장 크기가 다를 것이라는 가설을 세웠습니다.

생리 데이터(동공 확장)와 행동 데이터(경로 효율성, 반응시간)를 연결해 맥락 인식 메커니즘을 검증하고자 했습니다.

문제

조건 간 평균 비교 분석 결과 유의한 차이가 나타나지 않았습니다. 선행연구와 반대되는 결과였습니다.

단순히 '결과 없음'으로 처리할 수도 있었지만, 데이터 구조를 다시 검토했을 때 근본적인 문제를 발견했습니다. '같은 맥락' 조건 안에서도 참가자가 환경을 완전히 동일하다고 인식한 경우와 미세하게 다르다고 인식한 경우가 섞여 있을 가능성이 있었습니다. 행동 데이터만으로는 이 두 경우를 구분할 수 없었습니다.

결국 집단 평균이 무의미할 수 있는 상황이었습니다. 조건 간 차이가 없는 게 아니라, 평균을 내는 과정에서 패턴이 지워지고 있었을 수 있었습니다.

사고 과정

집단 평균 비교로는 이 문제를 해결할 수 없다고 판단했습니다. 분석 프레임을 전환했습니다.

'조건 간 차이'가 아닌 '개인 간 차이'로 접근했습니다. 동공 확장량이 큰 사람과 작은 사람이 행동적으로도 다른 패턴을 보인다면, 동공 신호가 실제로 맥락 인식을 반영한다는 것을 간접적으로 검증할 수 있었습니다.

개인 간 절대적 동공 크기 편차를 제거하기 위해 참가자별 z-scoring을 적용한 후 재분석했습니다.

해결

조건별 동공 확장량(z-scored)과 행동 지표 간 상관 분석을 수행했습니다:

  • Same context 조건: 동공 확장량 vs 경로 효율성 / RT 효율성
  • Different context 조건: 동공 확장량 vs sigmoid model fitting 정확도

Same / different 조건을 분리해서 분석한 이유는, 두 조건에서 동공 확장이 다른 인지 과정을 반영할 수 있기 때문이었습니다.

결과

  • Same context 조건에서 동공이 더 확장된 사람일수록 전반적으로 경로 효율성이 높은 marginal trend를 확인했습니다
  • Different context 조건에서 동공이 더 확장된 사람일수록 sigmoid model fitting이 잘 되지 않는 marginal trend를 확인했습니다
  • 집단 평균 비교에서 보이지 않던 개인차 패턴이 분석 프레임 전환 후 드러났습니다

실무 연결

집단 평균이 유의하지 않아도 개인차 수준에서 의미 있는 패턴이 존재할 수 있습니다. 서비스 로그에서도 전체 사용자 평균으로는 보이지 않는 신호가 사용자 세그먼트별 분석에서 드러나는 경우와 구조적으로 동일합니다.

생리 데이터(동공)와 행동 데이터(경로 효율)라는 서로 다른 종류의 데이터를 연결해 단일 지표로는 포착되지 않는 패턴을 식별한 경험은, 다양한 형태의 서비스 데이터를 통합해 분석하는 방식에 적용할 수 있다고 생각합니다.
