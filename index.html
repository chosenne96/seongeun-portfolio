Measuring the Unmeasured: Designing Spatial Navigation Metrics

Existing metrics captured movement efficiency but missed how people sample space under ambiguity. Designed novel behavioral metrics (path entropy, rotational complexity, speed-based decision indicators) from VR navigation logs. MATLAB Physiological Data Z-scoring Individual Differences

When Averages Lie: From Group to Individual Analysis

Group-level comparison of pupil dilation across context conditions showed no significant effect — contradicting prior research. Diagnosed the issue: averaging masked individual variation. Shifted the analysis frame from 'condition differences' to 'individual differences,' applied within-subject z-scoring, and linked pupil responses with behavioral patterns (path efficiency, model fitting accuracy).

R Signal Detection Theory Statistics

Verifying Decision Bias with Data: A Change Detection Study

Tested whether anomaly detection accuracy shifts when base rates change (20%/50%/80%). Applied signal detection theory (d', c) to separate genuine sensitivity changes from criterion shifts. Found that miss/false alarm patterns diverged significantly only at extreme base rate differences, and defined explicit outlier criteria before analysis to prevent post-hoc bias.

2023 | CAU Psychology Academic Festival
MATLAB R Preprocessing Z-scoring

When Results Don't Show: How Preprocessing Changes Analysis

Condition-level comparison of subjective time estimates showed no effect. Identified that individual baseline differences in estimation scale were masking the signal. Applied within-subject z-scoring to remove between-subject variance while preserving within-subject patterns — revealing significant condition effects that raw scores had obscured.
